## 技术栈

- 框架：Gin
- ORM：Gorm
- 数据库：MySQL

## 目录结构

```bash
├─controller  // 控制层
├─dao		  // 数据访问层
├─models	  // 模型
├─routers	  // 路由层
├─statics	  // 静态文件
└─templates	  // 网页模板
```
